# Job Portal

A simple **Job Portal** web application built using **HTML, CSS, and JavaScript**. The application allows users to log in, search for jobs, save their favorite jobs, and apply for available job openings.

## Features

- User Login Authentication
- View Available Jobs
- Search Jobs
- Save Jobs
- Remove Saved Jobs
- Apply for Jobs
- Responsive Design
- Local Storage Support

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
